How to create a new section in the website?

There are 2 types of sections:

  1. With Individual Page: Requires 2 extra steps to set up.
  2. Without Individual Page

Common steps for both types of sections

Step 1: Creating the _sectionname folder- This folder will have all the data of every page. Will be populated by MD files generated through the forms.

Step 2: Creating the sectionname.html file- This page loops through all the data about a particular section and displays it.

{% for variable_name in site.section_folder %} // To display any field value, use {{ variable_name.field_name }} // To link to the individual page, use {{ variable_name.url }} {% endfor %}

Extra steps for sections with individual pages

Step 3: Setting up the _config.yml file- This means that the page has an individual part as well.

Section_name: output: true

Step 4: Creating the layout file- In order to display the individial pages, need to be inside _layouts folder.

--- layout: Section layout type --- . . HTML Code Use {{ page.field_name }} to display the value of any field. . .
